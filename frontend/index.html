📖 Docs, Setup & Manual Testing

Everything you need to integrate, test and extend the GraphRAG platform.

🚀 Quick Start

Run the complete stack in 3 steps:

# 1. Clone and setup git clone https://github.com/PAMIDIROHIT/-Knowledge-Graph-Augmented-Retrieval-System.git cd Knowledge-Graph-Augmented-Retrieval-System/graphrag # 2. Start infrastructure (Neo4j, MLflow, monitoring) docker-compose up -d neo4j mlflow prometheus grafana # 3. Start the FastAPI server source .venv/bin/activate uvicorn src.api.main:app --reload --port 8000 # Open: http://localhost:8000 # Open this frontend: open frontend/index.html

🔑 Environment Variables

Copy .env.example → .env and fill in:

🧪 Manual Testing — Step by Step

Test each component independently:

# ── Unit tests (61 tests) ────────────────────────── pytest tests/ -v # ── Run the full 4-stage pipeline ────────────────── # Put your documents in data/raw/ dvc repro # ── Or run stages individually ───────────────────── python pipelines/ingest.py --config params.yaml python pipelines/extract.py --config params.yaml python pipelines/build_graph.py --config params.yaml python pipelines/index.py --config params.yaml # ── Test API manually with curl ──────────────────── # Health check curl http://localhost:8000/health | python -m json.tool # Query (local strategy) curl -X POST http://localhost:8000/query \ -H "Content-Type: application/json" \ -d '{"question":"What are the key entities?","search_mode":"local"}' \ | python -m json.tool # Graph explore curl "http://localhost:8000/graph/explore?entity_name=Microsoft&depth=2" \ | python -m json.tool # ── Run evaluation benchmark ─────────────────────── python eval/benchmark.py python eval/multihop_qa.py # quality gate: must score ≥ 0.70

🔗 Python Integration Example

Use the API from any Python backend:

import requests API = "http://localhost:8000" # Simple query resp = requests.post(f"{API}/query", json={ "question": "What companies are related to AI research?", "search_mode": "hybrid", "top_k": 5 }) data = resp.json() print(data["answer"]) for c in data["citations"]: print(f" [{c['source']}] {c['text'][:80]}…")
