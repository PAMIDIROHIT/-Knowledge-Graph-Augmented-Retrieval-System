The Transformer architecture was introduced in the 2017 paper "Attention Is All You Need" by Vaswani, Shazeer, Parmar, Uszkoreit, Jones, Gomez, Kaiser, and Polosukhin at Google Brain. The Transformer replaced recurrent neural networks with self-attention mechanisms, enabling significantly faster training through parallelization. The architecture consists of an encoder and decoder, each composed of stacked layers of multi-head attention and feed-forward networks.

BERT (Bidirectional Encoder Representations from Transformers) was developed by Google AI Language and introduced in 2018. BERT uses the encoder portion of the Transformer and is pre-trained using masked language modeling and next sentence prediction. BERT achieved state-of-the-art results on eleven natural language processing tasks when released. Google deployed BERT to improve Google Search query understanding in 2019. Jacob Devlin led the BERT development team at Google.

GPT (Generative Pre-trained Transformer) was developed by OpenAI. GPT-1 was introduced in 2018. GPT-2 was released in 2019 with 1.5 billion parameters and was initially not fully released due to concerns about misuse. GPT-3 was released in 2020 with 175 billion parameters and demonstrated remarkable few-shot learning capabilities. The GPT series uses the decoder portion of the Transformer architecture trained with causal language modeling. Alec Radford was the lead researcher on the GPT papers at OpenAI.

Retrieval-Augmented Generation (RAG) was introduced in the 2020 paper by Patrick Lewis, Ethan Perez, Aleksandra Piktus, Fabio Petroni, Vladimir Karpukhin, Naman Goyal, Heinrich Küttler, Mike Lewis, Wen-tau Yih, Tim Rocktäschel, Sebastian Riedel, and Douwe Kiela from Facebook AI Research. RAG combines parametric memory (language model weights) with non-parametric memory (retrieval from a document store). RAG retrieves relevant documents using a dense retriever (DPR) and then generates answers using a seq2seq model. RAG reduces hallucination by grounding responses in retrieved evidence.

GraphRAG was developed by Microsoft Research to address limitations of standard RAG. Standard RAG struggles with queries that require reasoning across multiple documents or understanding themes and patterns across a corpus. GraphRAG builds a knowledge graph from source documents by extracting entities and relationships, then uses community detection to identify themes. At query time, GraphRAG searches the knowledge graph and community summaries in addition to vector embeddings. The GraphRAG paper was authored by Darren Edge, Ha Trinh, Newman Cheng, Joshua Bradley, Alex Chao, Apurva Mody, Steven Truitt, and Jonathan Larson from Microsoft.

FAISS (Facebook AI Similarity Search) is a library for efficient similarity search and clustering of dense vectors. It was developed by Facebook AI Research and released in 2017. FAISS supports billion-scale vector search. It provides multiple index types including HNSW (Hierarchical Navigable Small World), IVF (Inverted File Index), and Flat indexes. FAISS is written in C++ with Python bindings. Jeff Johnson and Matthijs Douze are key contributors to FAISS. FAISS uses inner product and L2 distance metrics for similarity computation.

The Leiden algorithm for community detection was published in 2019 by Vincent Traag, Ludo Waltman, and Nees Jan van Eck. Leiden improves upon the Louvain algorithm by guaranteeing well-connected communities. It works by iteratively moving nodes to neighboring communities to maximize modularity, then aggregating communities into meta-nodes. Leiden is computationally efficient and produces better quality communities than Louvain for large graphs. The graspologic library implements the Leiden algorithm in Python.

LangChain is an open-source framework for building applications powered by language models. It was created by Harrison Chase and released in October 2022. LangChain provides abstractions for chains of LLM calls, retrieval-augmented generation, agents, and memory. LangChain is available in Python and JavaScript. The framework has become one of the most popular tools for LLM application development with over 80,000 GitHub stars. LangChain raised $25 million in a seed round in April 2023 led by Benchmark.

LlamaIndex (formerly known as GPT Index) is a data framework for connecting custom data sources to large language models. It was created by Jerry Liu and released in November 2022. LlamaIndex provides tools for data ingestion, structuring, retrieval, and integration with LLM applications. LlamaIndex supports hundreds of data connectors and integrates with LangChain and other frameworks. LlamaIndex raised $8.5 million in a seed round in 2023.

Hugging Face is a company that builds tools for machine learning and hosts the largest open-source model hub. It was founded in 2016 by Clément Delangue, Julien Chaumond, and Thomas Wolf. Hugging Face is headquartered in New York City with offices in Paris. The Transformers library by Hugging Face is the most widely used NLP library with over 100,000 pre-trained models. Hugging Face raised $235 million in a Series D round in 2023 at a valuation of $4.5 billion. The company hosts the Hugging Face Hub with over 500,000 models, datasets, and demos.
